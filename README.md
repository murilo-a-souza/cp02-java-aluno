## Objetivo
A classe aluno recebe os dados RM, Nome e Data de Nascimento e calcula a idade de cada objeto (aluno) e retorna.

Usamos construtores vazios para dois objetos que são definidos pelo usuário e dois com passagem de parâmetros (definidos por nós mesmos, a pedido do professor). 

Para fazer funcionar usamos classes como LocalDate para pegar datas e impedir o usuário de digitar datas além de um limite pedido, DataTimeFormatter para permitir o uso de datas no formato brasileiro, JOPtionPane para qualquer mensagem e interação e sempre se atentados a tratar error com try-catch.
